了解一下

我们希望你利用BillHub，完成组织内的所有收款渠道掌握、对账

1.我们的系统能为你能做什么？​

我们主要提供着4个功能： 统一对账，流水，回单管理，支付

场景一： 上我系统，做一次配置后，你可以管理所有的收钱渠道和对账了，后续权限在系统管理方，其他地方收钱给开个权限就好了。 不管是有系统没系统都行。

场景二： 自动化对账，以前1个人搞半个月或者多久的工作量，我系统完成了，你就可以安排这个人干别的活了，比如说一个月给你省1000，一年省1万， 而且人力宝贵，海不只是1万

场景三： 数据留存，后续审计查询数据能快速调出来

2.你需要提前做什么​

  • Docker:
    • 安装 Docker 和 Docker Compose

基于docker完成本地化的程序部署

3.运行你的服务器​

运行以下命令后，经典模板将自动添加到您的项目中：

docker-compose build && docker-compose up -d

命令在本地构建您的网站，并通过开发服务器提供服务，供您在http://localhost:3000/.
